# Dark Gaming Studio

Welcome to **Dark Gaming Studio**.

Founded in **2020 (1399 Hijri)**, we are a private game development team focused on building high-quality gaming experiences, server infrastructure, and developer tools. While most of our work is developed privately, we also publish selected open-source projects for the community.

## What We Do

Our work covers a wide range of game development and server technologies, including:

* 🎮 Game Development
* 🖥️ Dedicated Game Servers
* ⚙️ Server Infrastructure
* 🔐 Security Research & Demonstrations
* 🛠️ Developer Tools
* 🌐 Backend Services & APIs

We actively develop for multiple platforms and games, including:

* VMP
* Rust
* Arma 3
* Multi Theft Auto (MTA)
* and many more.

## Technology Stack

Our primary technologies include:

* C#
* .NET
* Lua
* PHP
* JavaScript
* SQL

## Our Mission

Our goal is simple:

> Build reliable software, improve the gaming community, and share useful tools and knowledge with developers whenever possible.

We believe that quality software, strong infrastructure, and open collaboration help create better gaming communities for everyone.

## Open Source

Although most of our internal projects remain private, we regularly publish selected repositories that can benefit developers and server owners.

If you find one of our projects useful, consider giving it a ⭐. Your support helps us continue developing and maintaining open-source software.
